# -Software-Design-Class-Project-Proposal-Proof-of-Concept

Instructions for running code:
Uses Python 3, most recent version is recommended.
Download from https://python.org
Library Dependencies: requests, BeautifulSoup4
Download libraries with pip (Windows):
pip install requests
pip install beautifulsoup4
If the above commands result in 'pip' not being a recognized command, try prefacing with both py -m or python -m, for example:
py -m pip install requests
